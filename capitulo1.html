O que é INFORMÁTICA?

Informática é a ciência que estuda as práticas de processamento, transmissão e armazenamento de dados por meios digitais. Possui a finalidade de resolver problemas, assim uma máquina é desenvolvida com o intuito de seguir uma série de instruções, chamadas de algoritmos para assim conseguir resolver o problema e, facilitar a vida das pessoas no dia a dia.

Na Informática há dois termos essenciais

HARDWARE

Se refere a parte física da informática, a parte que pode ser tocada, sendo equipamentos como: placas de vídeo, impressoras, notebooks, teclados, mouses etc.

SOFTWARE

Se refere a parte intangível da informática, os famosos programas de computadores, aplicativos de celulares e websites.

"O faz Tudo"

  • Planejar a construção de um sistema/software

Linguagens de programação


"As ferramentas"

Para se criar algo precisa-se de ferramentas certo? Pois bem, as ferramentas dos desenvolvedores para a criação de sistemas são as linguagens de programação. Linguagens de programação são um meio de contato entre computadores e os seres humanos, elas são com a “língua” que os computadores entendem

Existem dois tipos principais de linguagens de programação, as de alo nível e as de baixo nível. As de baixo nível são linguagens com sintaxe, ou seja o jeito de escrever, mais próximo ao da máquina (zeros ou um), sendo extremamente difíceis para interpretação humana. Do outro lado, sendo totalmente o oposto das linguagens de baixo nível, temos as linguagens de alto nível, que são mais próximas da linguagem humana e, portanto mais simples para compreensão.

Tipos de_Desenvolvedores

Desenvolvedor Front End

É o desenvolvedor que cuida da interface do usuário, ou seja, a parte da frente do sistema, onde ficam os botões, campos de texto e etc...

Desenvolvedor Back End

É o desenvolvedor que atua atrás das cortinas, é ele quem cuida de partes como autenticações, pagamentos e outras funcionalidades não visíveis para o usuário.

Desenvolvedor Full Stack

Os desenvolvedores Full Stack possuem conhecimento sobre os dois tipos de desenvolvimento, portanto, são aqueles que podem atuar tanto no Back End como no Front End.
